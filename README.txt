Wiring:
	- LCD: 
		https://howtomechatronics.com/tutorials/arduino/lcd-tutorial/
		Check class for pins used
		Contrast resistance: 9.1-9.4 Kohms

	- TRC:
		Arduino		TRC
		   (20) SDA-SDA
		   (21) SCL-SCL

	- DHT11:
		Arduino		DHT11
		        D31-Signal

	- Proximity:
		Arduino		Proximity Sensor
		        D22-Signal

Libraries to install on Arduino IDE:
	- RTClib
	- DHTlib
